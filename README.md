# LeastAction

Principle of Least Action Interactive — An educational demonstration of the principle of least action in physics.

Originally created by Slavomir Tuleja & Edwin F. Taylor (2003) as Java applets. Modernized from Java applets to JavaScript.

Original source: https://www.eftaylor.com/software/ActionApplets/LeastAction.html
